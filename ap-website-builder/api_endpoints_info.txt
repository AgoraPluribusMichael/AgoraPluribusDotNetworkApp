Plugin Management:
- GET /api/v1/plugins - List available plugins

Speech-to-Text (STT) Endpoints:
- GET /api/v1/plugins/stt/recording/start - Start audio recording
- GET /api/v1/plugins/stt/recording/stop - Stop recording and get transcription

Site Management:
- GET /api/v1/sites - List all sites
- POST /api/v1/sites - Create new site
- GET /api/v1/sites/<site_id> - Get specific site
- PUT /api/v1/sites/<site_id> - Update site
- DELETE /api/v1/sites/<site_id> - Delete site

Page Management:
- GET /api/v1/sites/<site_id>/pages - List pages for a site
- GET /api/v1/sites/<site_id>/pages/<page_id> - Get specific page content
- POST /api/v1/sites/<site_id>/pages - Create new page
- PUT /api/v1/sites/<site_id>/pages/<page_id> - Update page content
- DELETE /api/v1/sites/<site_id>/pages/<page_id> - Delete page

CORS Support:
- OPTIONS /api/v1/<path:path> - Handle preflight requests

The server runs on 127.0.0.1:8000 and restricts access to localhost only.
